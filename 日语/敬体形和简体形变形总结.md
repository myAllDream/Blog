- **动词的变形**

|            |      | 敬体形           | 简体形         | 敬体形           | 简体形   |
| ---------- | ---- | ---------------- | -------------- | ---------------- | -------- |
| 现在将来时 | 肯定 | 買います         | 買う           | あります         | ある     |
|            | 否定 | 買いません       | 買わない       | ありません       | ない     |
| 过去式     | 肯定 | 買いました       | 買った         | ありました       | あった   |
|            | 否定 | 買いませんでした | 買わない買った | ありませんでした | なかった |
